Projetos Destacados



Front-end desenvolvido com Angular, Bootstrap e CSS personalizado, para o sistema de gestão educacional, no qual professores e pedagogos são os usuários finais. Layout com dashboard de cards dinamicos e gráfico com Chart.js. Principais funcionalidades: Cadastro, edição, exclusão e consulta de alunos, professores, pedagogos, exercícios, avaliações e atendimentos pedagógicos. Restrições de funcionalidades para cada classe de usuário logado, autenticação de usuários por Token JWT, consumo da API ViaCEP para cadastro de endereços e validação dos campos de todos os formulários. 100% Responsivo. API REST desenvolvida com C# .NET, EntityFramework para ORM, AutoMapper, Swagger para documentação e SQL Server, com as tabelas criadas por Code First. Data Validations para todas as classes. Segui boas práticas de Clean Code e arquitetura SOLID.

Monitoramento de indicadores de saúde através de gráficos interativos e intuitivos, utilizando a biblioteca Chart.js. Desenvolvimento de dashboards personalizados com Bootstrap e CSS personalizado que oferecem visualizações em tempo real de métricas como frequência cardíaca, níveis de glicose, pressão arterial e atividade física, permitindo aos usuários acompanhar progressos e detectar tendências em seus dados de saúde. Conta com sistema de autenticação de login por Token JWT, reforçando a privacidade e proteção dos dados dos usuários. 100% Responsivo. API REST desenvolvida com C# .NET, EntityFramework para ORM, AutoMapper, Swagger para documentação e SQL Server, com as tabelas criadas por Code First. Data Validations para todas as classes. Segui boas práticas de Clean Code e arquitetura SOLID.

No front-end, utilizei Angular juntamente com Bootstrap e CSS personalizado para construir uma interface de usuário dinâmica e responsiva. Para a segurança e autenticação, implementei guards de rotas em Angular, assegurando que apenas usuários autenticados acessassem áreas específicas do aplicativo, e criei serviços específicos para a administração. A geolocalização foi um elemento crucial, para o qual integrei a API do ViaCEP para preenchimento automático de endereços e a API do LocationIQ, exibindo mapas interativos através da biblioteca Leaflet. As validações de formulários foram aprimoradas usando uma abordagem Template-Driven com Angular, proporcionando validações integradas e um serviço de autenticação robusto. A estruturação do projeto em Angular visou eficiência e escalabilidade, utilizando o HttpClient para consumir APIs de forma eficiente. Além disso, desenvolvi uma interface amigável, com uma Homepage destacando os "Lanches do Dia", e módulos para Categorias, Produtos, Carrinho, e Histórico de Compra. No módulo de Administrador, criei um dashboard para gerenciamento completo de produtos, pedidos e categorias, facilitando a criação de promoções e combos.
